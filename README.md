# Self-Driving Car Simulation 🚗🤖

This project simulates autonomous cars learning to drive on a track using neural networks and a genetic algorithm. Each car has independent sensors and a neural network, evolving over generations to improve performance.

## Features
- Independent car agents with their own neural networks 🧠
- Sensors (raycasting) for detecting track boundaries 📡
- Steering and braking controlled by neural networks ⚙️
- Genetic algorithm for selection, mutation, and elitism 🧬
- Fitness function based on distance traveled, stability, and path tracking 📈
- Crash handling: cars that hit walls are removed (no effect on others) 💀

## Tech Stack
- Python 🐍
- NumPy 🔢
- Pygame 🎮
- Custom ML logic 🧠⚡
